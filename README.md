# Shopping List

Android-приложение для управления списком покупок. Позволяет добавлять, редактировать и удалять товары, отмечать их как купленные. Поддерживает планшеты и телефоны.

# Стек технологий

Kotlin

MVVM + Clean Architecture — разделение на слои data, domain, presentation
Dagger 2 

Room 

ContentProvider 

DataBinding 

LiveData 

Coroutines

ViewBinding 

RecyclerView + ListAdapter + DiffUtil 

ItemTouchHelper 

Адаптивный UI
